Semantic HTML

Element Placement

Tag Description
<header> A heading or title area of a page.
<nav> A pages navigational links.
<main> The main content of a page between header and footer.
<footer> Below the main content.

Embedding Media

Tag Description
<video> Add videos to a web page using src.
<audio> Add audio to a web page using src.
<embed> Add any type of media file using src. No closing tag.

Figure and Figcaption

Tag Description
<figure> Encapsulate media.
<figcaption> Descibre media.

Section and Article

Tag Description
<section> Related areas of a page.
<article> Detailed information within a section.

Aside

Tag Description
<aside> Additional information to enhance.
